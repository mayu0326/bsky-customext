Bluesky User Menu 拡張機能 プライバシーポリシー

本拡張機能は、Bluesky 上でフォロー・ブロック・リスト追加を行うために、以下の情報をブラウザ内に保存します。
・Bluesky のハンドル名
・Bluesky の App Password（アプリパスワード）

これらの情報は、拡張機能が Bluesky の公式 API にアクセスする際の認証にのみ使用されます。
収集した情報を、開発者または第三者のサーバーへ送信することはありません。

また、本拡張機能はBluesky の Webページ内容を読み取り、
ユーザークリックを検出してメニューを表示しますが、
閲覧したページの内容や履歴を外部に送信することはありません。

本拡張機能をインストール・利用した時点で、本ポリシーに同意したものとみなします。
