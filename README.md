# Bluesky User Menu

- Blueskyのユーザー名をクリックしたときに、フォロー・ブロック・リスト追加などのメニューを表示して、\
ワンクリックで操作できるようにするChrome拡張機能です。

## 機能

- Blueskyのタイムラインやプロフィール上のユーザー名をクリックすると専用メニューを表示
- メニューからそのまま「フォロー」を実行
- メニューからそのまま「ブロック」を実行
- ユーザーを任意のリスト（拡張機能内で管理するリスト）に追加予定

※Blueskyの公式API（AT Protocol）を利用してフォロー／ブロックを行います。

## インストール方法（開発版）

1. このリポジトリをクローンまたはダウンロードします。
2. Chromeで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」をクリックし、このフォルダを選択します。
## 使い方

1. 拡張機能のオプションページを開き、以下を設定します。
   - Blueskyのハンドル名（例: `hoge.bsky.social`）
   - BlueskyのApp Password（アプリパスワード）
2. BlueskyのWebクライアント（`https://bsky.app/`）を開きます。
3. タイムラインやプロフィールのユーザー名をクリックします。
4. 画面上に表示されるメニューから「フォロー」「ブロック」「リストに追加」などをクリックすると、その操作が実行されます。

## 権限とデータの取り扱い

この拡張機能は、以下の権限とデータを利用します。

- `storage`  
  - Blueskyのハンドル名とApp Password、およびリスト設定をブラウザ内に保存するために使用します。
- `https://bsky.app/*`  
  - Blueskyのページ上でユーザークリックを検出し、メニューを表示するために使用します。
- `https://bsky.social/*` / `https://public.api.bsky.app/*`  
  - Blueskyの公式APIにアクセスし、フォロー／ブロック処理を行うために使用します。

収集した情報（ハンドル名・App Passwordなど）は、拡張機能がBluesky APIへアクセスするための認証にのみ使用され、開発者や第三者のサーバーへ送信されることはありません。

詳細はプライバシーポリシーをご覧ください。
